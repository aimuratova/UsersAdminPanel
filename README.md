# UsersAdminPanel

Кратко
: Панель администрирования пользователей на .NET 8 (Razor Pages + минимальные API/контроллеры). Проект предназначен для управления пользователями (блокировка, удаление, просмотр) и интегрируется с существующими сервисами в решении.

Требования
- .NET 8 SDK
- Visual Studio 2022/2023 (рекомендуется) или `dotnet` CLI
- База данных / конфигурация (указана в `appsettings.json` или в переменных окружения)

Структура проекта
- `Pages/` — Razor Pages интерфейс (основной UI).
- `Controllers/` — API/контроллеры для AJAX-операций (напр., `UsersController`).
- `Services/` — бизнес-логика (интерфейсы и реализации)
- ссылка на проект DAL для доступа к данным пользователей.
- `wwwroot/` — статические файлы (CSS, JS, изображения).
- используется Bootstrap для стилизации и jQuery для AJAX-запросов.
- внедрение зависимостей для сервисов и репозиториев в `Program.cs`.
- аутентификация и авторизация используя JWT или cookie-based механизмы (настраивается в `Program.cs`).

Быстрый старт
1. В Visual Studio откройте решение и выберите проект как стартовый.
2. Настройте строки подключения в `appsettings.Development.json`.
3. Запустите в режиме отладки (F5) либо
